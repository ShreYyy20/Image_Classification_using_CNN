# **Image Classification Using CNN with CIFAR-10 Dataset**
This repository contains a Jupyter Notebook demonstrating the implementation of a Convolutional Neural Network (CNN) for image classification using the CIFAR-10 dataset. The tutorial includes data preparation, model building, training, and evaluation using Tensorflow,Keras and Scikit-learn.
# **Summary**
This tutorial explores the process of building and training a CNN for image classification using the CIFAR-10 dataset. The CIFAR-10 dataset consists of 60,000 32x32 color images in 10 different classes. The tutorial showcases the implementation of a CNN model in TensorFlow and Keras, including data preparation, model architecture, and performance evaluation.

# Highlights
- **Imports:** Import necessary libraries and packages for data handling, model building, and visualization.
- **Load Datasets:** Load and prepare the CIFAR-10 dataset.
- **Observe Dataset:** Visualize and understand the structure of the dataset.
- **Normalize Data:** Normalize the image data to improve model performance.
- **Building Model:** Construct the CNN model with convolutional and pooling layers.
- **Compile Model:** Compile the model with appropriate loss function, optimizer, and metrics.
- **Train the Model:** Train the model using the training dataset and validate it using the test dataset.
- **Making Predictions:** Use the trained model to make predictions on the test dataset.
- **Evaluate Model:** Evaluate the model's performance using accuracy and loss metrics on the test dataset.

# Project Highlights
- **Data Preprocessing:** Load the CIFAR-10 dataset, normalize the image data, and split it into training and test sets.
- **Model Building:** Construct a CNN model using TensorFlow and Keras. The model includes convolutional layers, max-pooling layers, flattening layers, and dense layers.
- **Model Training:** Compile the model with appropriate loss function, optimizer, and metrics. Train the model using the training data and validate it using the test data.
- **Model Evaluation:** Evaluate the model's performance using accuracy and loss metrics on the test dataset. Visualize the training history to understand the model's learning process.
- **Sample Predictions:** Make predictions on the test dataset and display sample images with their predicted labels.
